## General Keywords
### Mode
#### Keyword type
String

#### Description
This keyword controls the running mode of the GPAW. Available options are:

* PW
* PW-GW
* EXX
* LCAO
* FD.

#### Default
PW

#### Example
Mode = 'PW'
